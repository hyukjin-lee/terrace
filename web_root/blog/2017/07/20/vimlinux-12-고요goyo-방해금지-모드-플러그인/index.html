[vim/Linux] 12. 고요(Goyo), 방해금지 모드 플러그인

이 플러그인은 아래 스크린샷처럼 텍스트에만 집중할 수 있게 해줍니다

.vimrc에 Plugin 'junegunn/goyo.vim' 를 추가하고 :PluginInstall 을 합니다.

.vimrc에 다음과 같이 추가합니다.

" Goyo
nnoremap <F4> :Goyo <CR>
function! s:goyo_enter()
  set noshowmode
  set noshowcmd
  set scrolloff=999
  set number
  set rnu
  " ...
endfunction
 
function! s:goyo_leave()
  set showmode
  set showcmd
  set scrolloff=999
  " ...
endfunction
 
autocmd! User GoyoEnter nested call <SID>goyo_enter()
autocmd! User GoyoLeave nested call <SID>goyo_leave()

<F4>를 누르면 Goyo mode로 토글할 수 있습니다.

Goyo mode에 들어가면 커서가 항상 화면 가운데에 옵니다. 이 기능은 평소에도 유용하게 사용할 수 있을 것 같아서 기본으로 set scrolloff=999 옵션이 적용되도록 했습니다.

" Basic Settings
colorscheme CodeSchool3
set termguicolors
syntax on
set number
set relativenumber
set hlsearch
set ignorecase
set tabstop=4
set softtabstop=4
set shiftwidth=4
set expandtab
autocmd FileType make setlocal noexpandtab
set scrolloff=999
